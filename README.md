# SSO

Welcome to the SSO (Single Sign-On) project! This project provides a secure and scalable single sign-on solution, enabling users to authenticate across multiple applications using a single set of credentials


## Installation

Clone the repository:
```bash 
git clone https://github.com/yberikov/sso.git
cd sso
```

Run the app:
```go
go run .
```

## Usage - Endpoints
```plain
/login: User login endpoint.
/register: User registration endpoint.
/edit: User data editing endpoint.

```
